## Kernel/VM探検隊＠つくば No3

[あたまのおかしい探検隊が、11年ぶりにつくばに帰ってきた](https://kernelvm.connpass.com/event/381371/)ようだ。

金曜に大学内で開催された。
私は 10 分枠で発表した。
[Veryl の IR で遊んでみる](https://slide.ryoga.dev/posts/veryl-css.html)とかいうタイトルで、中身は Veryl to CSS をするというものだった。
時間がなく、スライドは徹夜して作成したがなんとかなった。
思ったよりもウケが良く、「ベスト・オブ頭おかしい賞」とかいうやつに選ばれたらしい。
HDL に触れてから CSS を書くと、どうも CSS の中の HDL みというかそういうものを感じていたので、HDL から CSS にコンパイルするという発想は自然なものだろうと思っていたが、実はそうではなかったらしい。

Twitter を眺めたりしていると反応があってよかった。

夜は、撤収作業の手伝いとかをした。
百香亭で懇親会があったが、人がギチギチで驚愕。
撤収作業で遅くなっていたこともあり、座るのが難しかった。
